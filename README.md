# Daraz Nepal Product Scraper & Analyzer 🛍️📊

A Python scraper for Daraz Nepal that extracts product name, price, rating, reviews, and product link.  
Exports cleaned data to CSV and generates beautiful bar charts & pie charts for price distribution, top brands, rating breakdowns, and review counts.  
Perfect for market research, price tracking, and competitive analysis.

**Features:**
- Scrape any product category (mobile, laptop, fan, etc.)
- Extract 5 key fields per product
- Export to CSV with one click
- Visualize data with 4 auto-generated charts (bar + pie)
- Built with Requests, BeautifulSoup, Pandas, Matplotlib, and Seaborn
- Runs in Jupyter Notebook

**Quick Start:**  
`pip install -r requirements.txt` → run the notebook → enter keyword & pages → get data + charts.
